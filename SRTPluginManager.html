SRT Plugin Manager Home

When you start up the program, the Home section is where you start off, being able to choose the other menus. On this tab you get some basic information.

×

SRT Plugin Manager Host

On the SRTHost Tab, you can choose the game you are currently playing and display informations that that Plugin offers. By pressing Install, you will download the current Plugin, if its outdated, you can press Update, to update it. Additionally you can install the SRTHost64 or SRTHost32 (Depending on the game you play). After you did that, go into Extensions, then back to start the SRTHost and then go to Widgets.

×

SRT Plugin Manager Extensions

In here you can install the needed Extensions to make the SRT work, I suggest to just install both of them.

×

SRT Plugin Manager User Interfaces

In User Interfaces, you install the ingame Overlays. They will be displayed over your game in real time.

×

SRT Plugin Manager Widgets

In Widgets you can deactivate certain Options to only show information you want to see. By copying the Stats URL, you can enter that into your Browser and track the Information via your browser. Inventory URL does the same, but its only showing your Inventory in real time (Only works for those games that offer it).
